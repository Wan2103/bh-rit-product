# Deployment Guide

## Overview

This guide explains how to deploy the BH RIT Product Presentation website.

The project is a static website built with:

- HTML
- CSS
- JavaScript
- JSON data files
- Static media assets

The website can be deployed using platforms that support static websites.

---

# Deployment Requirements

Before deployment, prepare:

- GitHub repository
- Hosting platform
- Domain name (optional)
- Production-ready files
