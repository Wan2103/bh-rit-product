# Content Guide

## Overview

The `content` system controls all text-based information displayed throughout the BH RIT Product Presentation website.

Content is separated from the website structure to make updates easier without modifying the main HTML files.

Content sources include:

- Product information
- Company information
- Industry information
- Services
- FAQs
- Testimonials
- Navigation content
